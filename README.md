# 口算能量訓練器 ｜ Math Power Timer

> 給國小孩子的每日心算練習小工具 —— 輕量、無壓力，幫助基礎計算更自動化。

一個單檔網頁 App，專為「會觀念、但基礎計算還不夠自動」的中年級孩子設計。
重點不是猛刷題，而是**每天短時間、低壓力**地練習，讓基本運算變得直覺，
這樣面對應用題時，working memory 才不會被計算卡住。

🔗 **線上試玩：** https://viviennelu712.github.io/math-power-timer/

---

## ✨ 特色

- **五種模式**：加法、減法、乘法、除法、混合
- **六個難度**：從 20 以內加減 → 進借位 → 九九乘法 → 表內除法 → 兩位數×一位數
- **不只看對錯，也看「卡頓」**：作答超過 4 秒的題目會被標記，找出真正不熟的計算類型
- **正向回饋**：Combo 連擊、能量點數、Mission Clear，像闖關遊戲一樣，但不喧鬧、不羞辱
- **結算報告**：正確率、平均速度、最高 Combo、錯題清單、慢題清單，可一鍵「重練錯題／重練慢題」
- **家長友善紀錄**：自動記錄近 7 次的平均正確率與速度，方便觀察孩子的進步
- **手機 / 平板 / 電腦皆可用**，響應式版面

## 🎯 設計理念

1. 短練習，不是重訓
2. 平靜但有動力的遊戲感
3. 不貼負面標籤、不製造考試壓力
4. 同時追蹤「正確率」與「反應速度（卡頓）」
5. 資料只存在本機（localStorage），不收集任何個資

## 🛠 技術

- 純 **HTML + CSS + 原生 JavaScript**，單一檔案，免安裝、免後端
- 紀錄使用瀏覽器 **localStorage** 保存
- 可加到手機 / 平板主畫面，像 App 一樣開啟

## 🚀 使用方式

直接打開上方的線上連結即可，或下載 `index.html` 用瀏覽器開啟。
在手機 / 平板的 Safari 開啟後，按「分享 → 加入主畫面」，桌面就會出現圖示，每天點開即可練習。

> 💡 紀錄存在各自的裝置裡、不會跨裝置同步，建議固定用同一台裝置練習，
> 家長看到的「近 7 次平均」才會連續。
